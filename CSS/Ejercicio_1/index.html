¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Nos permite estructurar el contenido de un sitio mediante etiquetas.

Etiquetas básicas

  • <h1>...</h1>: Títulos y encabezados.
  • <p>...</p>: Párrafos de texto.
  • <a>...</a>: Enlaces a otras páginas.
  • <img>: Imágenes.
