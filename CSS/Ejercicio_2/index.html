Introducción

En este curso aprenderás los fundamentos de HTML, el lenguaje básico para crear sitios web. Verás cómo usar etiquetas, atributos y la estructura básica de un documento HTML5.

Temas del curso

Módulo Contenido Duración
1 Introducción a HTML 2 horas
2 Estructura de una página 3 horas
3 Etiquetas y atributos 3 horas
4 Enlaces e imágenes 2 horas

Ejemplo básico en HTML

<!DOCTYPE html>
<html>
    <head>
        <title>Mi primera página</title>
    </head>
    <body>
        <h1>Hola Mundo</h1>
        <p>Este es mi primer sitio web</p>
    </body>
</html>
